# Day 2 - Linux Practice

## Commands Practiced

- sudo -i : Switch to the root user.
- logout : Log out of the current session.
- exit : Exit the current shell or terminal.
- sudo su : Switch to the root user.
- lscpu : Display CPU information.
- lsmem : Display memory information.
- df -h : Display disk space usage in a human-readable format.
- clear : Clear the terminal screen.
- hostname -i : Display the IP address of the host.
- date : Display the current date and time.
- echo "text" : Print text to the terminal.
- ls -a : List all files, including hidden files.

## What I Learned
Today, I practiced Linux system administration and system information commands.
I learned how to switch users, check CPU and memory details, view disk usage, display the system IP address, check the current date and time, print text to the terminal, and list hidden files.
